## 测试内容

### Alpha #1 因子公式
```
rank(Ts_ArgMax(SignedPower(((returns < 0) ? stddev(returns, 20) : close), 2.), 5)) - 0.5
```

### 测试流程
1. **数据获取**: 使用baostock获取真实股票数据
2. **因子计算**: 计算Alpha #1因子值
3. **策略回测**: 基于因子值构建投资组合
4. **结果分析**: 计算收益率、夏普比率等指标
5. **可视化**: 绘制累积收益曲线和分析图表

## 测试参数

### 默认设置
- **数据源**: baostock (免费)
- **测试期间**: 最近1年
- **股票池**: 沪深300成分股 (限制30只以提高速度)
- **选股数量**: 10只
- **调仓频率**: 5个交易日
- **权重分配**: 等权重

### 可调整参数
- `max_stocks`: 最大测试股票数量
- `top_n`: 选择的股票数量
- `rebalance_freq`: 调仓频率
- `start_date` / `end_date`: 测试时间范围

## 性能指标

测试程序会计算以下指标：

### 收益指标
- **总收益率**: 整个测试期间的总收益
- **年化收益率**: 年化后的收益率
- **超额收益**: 相对于基准的超额收益

### 风险指标
- **年化波动率**: 收益率的年化标准差
- **最大回撤**: 最大的资产回撤幅度
- **夏普比率**: 风险调整后的收益指标

### 其他指标
- **胜率**: 正收益交易日的比例
- **调仓次数**: 总的调仓次数

## 注意事项

### 数据源限制
- baostock是免费数据源，可能有访问频率限制
- 建议在非交易时间进行测试以获得更好的数据质量
- 如果遇到网络问题，可以减少测试股票数量

### 计算要求
- Alpha #1因子需要至少25天的历史数据
- 建议测试期间不少于6个月
- 股票数量过多会显著增加计算时间

### 结果解读
- 测试结果基于历史数据，不代表未来表现
- 因子有效性可能随时间变化
- 建议结合其他因子和风险管理措施
